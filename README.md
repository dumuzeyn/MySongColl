# Моя коллекция музыки



## Установка на ПК



1) Выберите папку в которую хотите загрузить музыку, и скопируйте ее имя `C:\Users\path\`

2) Нажмите `win r`, в окне напишите `cmd`

3) В открывшемся окне напиши `cd C:\Users\path\`

3) Далее напишите `git clone https:/github.com/dumuzeyn/MyMusicCollection` и нажмите `enter`



После этих дейсвий в папку `C:\Users\path\` установиться репозиторий в которой будет музыка.



## Установка на Телефон



Способ указанный ниже работает и на ПК.



1) Нажмите на кнопку `Code`

2) Появится два окна `Local` и `Codespaces` автоматически открывается `Local`

3) Снизу окна будет кнопка `Download ZIP`, нажмите и скачается `zip` файл



После этих действий сможете разархивировать `zip` файл в любую папку и слушать музыку. 

Если вы до этого скачивали `zip` файл а потом добавились новые песни в списке, то вам надо нажать на коммит в котором добавили новые песни и в ручную их устанавливать.
